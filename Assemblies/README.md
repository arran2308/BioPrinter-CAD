## Files

- `.step` – Universal CAD format for viewing and integration  
- `.f3z` – Native Fusion 360 archive  

## Usage

These files are intended for:

- Viewing the complete system layout  
- Understanding subsystem relationships  
- Importing into CAD software for reference  

For modification or manufacturing, refer to the **Subassemblies** and **Printed Parts** folders.
