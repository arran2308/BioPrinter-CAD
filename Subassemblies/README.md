# Subassemblies

This folder contains the major subsystems that make up the BioPrinter.

Each subassembly is provided as a separate CAD file to simplify:

- Design modification  
- System understanding  
- Assembly workflow  

---

The BioPrinter is composed of the following primary subsystems:

## Usage

Each subassembly can be opened independently in CAD software for:

- Detailed inspection  
- Modification and development  
- Integration into custom configurations  

For manufacturing, refer to the **Printed Parts** folder.
